Branching

Example with if-statements

 
            num = input("Enter a number: ")
            if num > 2: 
                print("You number is greater than 2")
            if num % 2 == 0: 
                print("Your number is even")

Example with mutually exclusive conditions

 
            num = input("Enter a number: ")
            if num > 0: 
                print("You entered a positive number")
            # elif is used for mutually exclusive conditions
            elif num < 0: 
                print("You entered a negative number")
            # everything else
            else: 
                print("You entered '0'")

Example with and operator

 
            num = input("Enter a number: ")
            if num >= 10 and num <= 15:
                print("Your number is between 10-15")
            else:
                print("Your number is less than 10 or greater than 15")

For Loops

Recall range() gets a range of numbers. You may pass in one, two, or three arguments. range(start, stop, step)

For loops can iterate through a range of numbers or a sequence of numbers.

for counter in range(10):
    print(counter)

Example iterating through a collection (list)

names = ["Jerry", "Tom", "Sammy"]
for each_name in names:
    print(each_name)

Functions

Example of defining and calling a function

Example of defining and calling a function with paramters

def function1(name, major):
    print("Hi my name is " + name + ", and I am studying" + major)
    
function1("Tommy", "Architecture") # calls the function

Example of defining and calling a function with default paramters

def function1(name="Tommy", major="Business"):
        print("Hi my name is " + name + ", and I am studying" + major)
        
function1() # calls the function with default arguments

Method What it does
list.append(some_item) Add an item to the end of the list.
list.insert(index, some_item) Insert an item (some_item) at a given position (index).
list.remove(some_item) Remove the first item from the list whose value is equal to some_item
list.pop() Remove the last item in list
list.clear() Removes all items from the list
list.sort() Sort the items in list
list.reverse() Reverse the items in list
