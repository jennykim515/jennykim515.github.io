Intro

Automated picture taker with Raspberry Pi 4 and a camera module. Picture is taken every hour and is stitched together to make a time-lapse video.

Demo

What I Learned

Setup: This is the first time I've used a raspberry pi. The most difficult part of the project was setting up. The lighting of the room had to be good at all hours of the day and the location should not move. I also learned how to set up the pi by hooking it up with a monitor, keyboard, and mouse. I worked only on the terminal to set up this project.

Cron: I learned how to use cron with this project. At one hour intervals, the pi took a picture and placed them in a folder. At the end, I was able to stitch a timelapse video together using a tutorial here.
