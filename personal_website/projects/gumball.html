Touchless Gumball Machine

The gumball machine is my favorite project to talk about because it interacts with the physical world and it was such a fun project to work on. It was my final project for ITP348: Intro to Electronics/Wearables. I made the contraption using a jigsaw and some wood I bought from Lowes, and I describe the process in the documentation below!



More on Github



Challengees

The most difficult challenge was the hardware/mechanical parts of the gumball machine. There were a lot of wires in a small space and a lot of sensors to include. After many trials and a new flashlight, I got it done!
